# 아이온2 숙제 관리 (Aion 2 Task Manager)

이 프로그램은 아이온2 게임을 플레이하는 유저들을 위한 숙제 및 자원 관리 도구입니다. 여러 캐릭터의 일일/주간 숙제 완료 여부와 자원 상태를 효율적으로 추적하고 관리할 수 있도록 돕습니다.

## 🚀 지금 바로 사용해보세요!

별도의 설치 과정 없이 웹 브라우저를 통해 접속하여 바로 사용할 수 있습니다.

**🔗 애플리케이션 접속 링크:** [https://simm4256.github.io/aion2-task-manager/](https://simm4256.github.io/aion2-task-manager/)

## ✨ 주요 기능

*   **캐릭터 관리:** 여러 캐릭터를 추가하고 이름을 편집할 수 있습니다.
*   **숙제 관리:** 일일/주간 숙제를 추가하고 캐릭터별 완료 여부를 체크할 수 있습니다.
*   **자원 관리:** 캐릭터별 자원(재료, 아이템 등)의 보유량을 추적하고, 충전 주기를 설정하여 자동으로 갱신할 수 있습니다.
*   **데이터 백업/복원:** '데이터 관리' 기능을 통해 현재 데이터를 백업하고 복원할 수 있습니다.

## 📝 간단한 사용 방법

1.  **애플리케이션 접속:** 위 링크를 클릭하여 웹 브라우저에서 프로그램을 엽니다.
2.  **PWA 설치 (선택 사항):** 브라우저 주소창에서 '앱 설치' 아이콘을 클릭하여 PWA로 설치하면, 바탕화면이나 시작 메뉴에서 바로 앱처럼 실행할 수 있습니다.
3.  **캐릭터 추가:** '숙제' 탭 또는 '자원' 탭에서 '캐릭터 추가' 버튼을 눌러 플레이하는 캐릭터들을 등록합니다.
4.  **숙제/자원 추가:**
    *   **숙제 탭:** '숙제 추가' 버튼을 눌러 일일/주간 숙제의 종류를 정의합니다.
    *   **자원 탭:** '자원 추가' 버튼을 눌러 추적하고 싶은 자원(예: 재료, 티켓)을 정의합니다. 충전 시간, 주기 등을 설정할 수 있습니다.
5.  **숙제 완료 체크:** '숙제' 탭에서 각 캐릭터별, 숙제별로 체크박스를 클릭하여 완료 여부를 표시합니다.
6.  **자원 수량 관리:** '자원' 탭에서 캐릭터별 자원 수량을 확인하고, 필요에 따라 수량을 직접 조절하거나 자동 충전을 이용합니다.
7.  **데이터 관리:** 우측 상단의 '데이터 관리' 버튼을 통해 현재 모든 데이터를 문자열로 내보내 백업하거나, 백업된 데이터를 불러와 복원할 수 있습니다.

이 프로그램으로 아이온2 숙제를 더욱 효율적으로 관리해 보세요!
