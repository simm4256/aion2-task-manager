# Aion 2 숙제 관리 실행 방법

이 애플리케이션을 로컬 컴퓨터에서 실행하고 사용하기 위한 단계별 안내입니다.

## 1. Node.js 설치

이 애플리케이션을 실행하려면 [Node.js](https://nodejs.org/)가 필요합니다. 컴퓨터에 Node.js가 설치되어 있지 않다면, 아래 공식 다운로드 페이지에서 **LTS (Long Term Support)** 버전을 다운로드하여 설치해주세요.

[**Node.js 공식 다운로드 페이지 바로가기**](https://nodejs.org/)

설치가 완료되면, 터미널(Windows의 경우 명령 프롬프트 또는 PowerShell)을 열고 다음 두 명령어를 각각 실행하여 설치가 올바르게 되었는지 확인합니다.

```bash
node -v
npm -v
```

각 명령어 실행 후 버전 번호(예: `v20.11.0`)가 표시되면 정상적으로 설치된 것입니다.

## 2. 프로젝트 설정

터미널에서 `cd` 명령어를 사용하여 이 프로젝트 폴더로 이동합니다.

```bash
# 예시: cd C:\Users\YourName\Desktop\aion2-task-manager
cd <프로젝트 폴더 경로>
```

그런 다음, 아래 명령어를 실행하여 프로젝트에 필요한 모든 라이브러리를 설치합니다. 인터넷 속도에 따라 몇 분 정도 소요될 수 있습니다.

```bash
npm install
```

## 3. 애플리케이션 실행

라이브러리 설치가 완료되면, 다음 명령어를 실행하여 애플리케이션을 실행합니다.

```bash
npm run preview
```

성공적으로 실행되면 터미널에 다음과 같은 메시지가 나타납니다.

```
  > Local: http://localhost:4173/
```

(`4173`은 다른 번호일 수 있습니다.)

## 4. 브라우저에서 열기

이제 웹 브라우저(Chrome, Edge 등)를 열고, 터미널에 표시된 `http://localhost:4173/` 주소를 주소창에 입력하여 접속합니다.

## 5. PWA(앱)로 설치하기

웹사이트에 접속하면 브라우저 주소창 오른쪽에 아래와 같이 생긴 '앱 설치' 아이콘이 나타납니다.

<img src="https://i.imgur.com/4g2VwMD.png" alt="PWA 설치 아이콘" width="400"/>

이 아이콘을 클릭하고 '설치' 버튼을 누르면, 웹사이트가 컴퓨터에 하나의 프로그램처럼 설치됩니다.

## 6. PWA로 실행

설치가 완료되면 바탕화면과 시작 메뉴에 'Aion 2 숙제 관리' 아이콘이 생성됩니다. 이제부터는 웹 브라우저를 열 필요 없이 이 아이콘을 더블클릭하여 프로그램을 바로 실행할 수 있습니다.
